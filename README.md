# Todoist Telegram Interface


## 1. Dependencies
- Python 3.8
- Poetry 1.1.13


## 2. Installation
### 2.1. Configuration
Create `.env` file and config your app
```bash
cp .env.template .env
```  
### 2.2. Install dependencies
```bash
poetry env use python3.8
poetry install
```

## 3. Start
```bash
poetry run python main.py
```
